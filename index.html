This is a version of Flying Bird for people with amblyopia, prepared to be used with glasses, over the ones you already wear, but with red and blue/cyan colored glasses (anaglyphs).

If you have a keyboard, the Graphics can be moved with .

Inspired by Dr. Mario Cerrella and his Visual Training, a group of Ophthalmologists, Orthoptists and Developers, decided to construct this and other games that allow "Lazy Eye" Treatment.

We need your help to follow this path. You can donate whenever you want writing to rafairibarren@gmail.com.

The source code of this game can be found here GitHub.
